Typography

is a little HTML and a lot of CSS

There are 4 typographic HTML elements that you will use most:

paragraphs headers spans line breaks

<p>Lorem ipsum dolor sit amet!</p>

Paragraphs are denoted by p, and are

containers for basic text

<h1>I ♥ Headers</h1>

Headers are denoted by h1, h2, h3, h4, h5, and h6, and are

containers for important text

<p>
    Lorem ipsum <span class="red">dolor sit amet</span>!
</p>

Span is denoted by span, and is used to

modify a specific section of text

within another typographic element

<p>Lorem ipsum<br />
dolor sit amet!</p>

Line Breaks are denoted by br, and are used to

force a new line

within another typographic element

of these 4, only

Line Breaks

cannot affect style

&

have no closing tags

a tag with no closing tags is called an empty tag

<p></p>

normal tag

vs

<br />

empty tag

body {
    font-size: 16px;
}

The font-size property defines

text size

(can be set in px, em, %, etc)

Check it out:

9-11 pt

print body copy

vs

13-16 px

web body copy

Many will argue for 13 px on the web as it is

more efficient and looks better

smaller type is often more visually appealing

Others prefer 16 px on the web as it is

much clearer and easier to read

bad legibility is one of the fastest ways to lose users

The em and % vs px battle

doesn't matter anymore

all modern browsers have normalized type sizing and scaling

px sizes are

absolute

(get what you see)

em and % sizes are

relative

(based on parent size)

body {
    color: #333333;
}

The color property defines

text color

(can be set in hex, rgb, rgba, etc)

Please, don't use pure black for text color

#333333

is much easier on the eyes

body {
    line-height: 1.125;
}

The line-height property defines

leading or vertical line space

(can be set in decimal, px, em, or %)

body {
    font-style: italic;
}

The font-style property defines

text styling

(commonly set to italic or oblique)

body {
    font-weight: bold;
}

The font-weight property defines

text strength

(can be set in normal, bold, or a number between 100 and 900)

body {
    text-decoration: underline;
}

The text-decoration property defines

text extras

(commonly set to underline or strike-through)

body {
    text-align: center;
}

The text-align property defines

horizontal position

(commonly set to left, right, center, or justify)

body {
    vertical-align: middle;
}

The vertical-align property defines

vertical position

(commonly set to top, bottom, middle, or baseline)

The text-align and vertical-align properties

only work on text elements

(later, we'll check out some magic to change this rule)

body {
    letter-spacing: 1;
}

The letter-spacing property defines

space between letters

(can be set in decimal, px, em, or %)

body {
    word-spacing: 1;
}

The word-spacing property defines

space between words

(can be set in decimal, px, em, or %)
