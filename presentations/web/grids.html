Let start with the grid column

<div class="column1">...</div>

Creating a class for the single column is simple:

class=“column1”

(remember, you can name the class anything you want)

Now to make the class meaningful, we'll create

.column1 {
    width: 60px;
    margin-left: 20px;
}

a CSS rule with a class selector and property definitions for

width & margin

body {
    width: 940px;
}

The width property defines

how wide an element will be

(can be set in px, em, or %)

Normally, containers resize to their contents, but

a set width will resize contents to fit a container

(setting width is like changing a container’s contents into water)

body {
    margin-top: 10px;
    margin-bottom: 10px;
    margin-left: 10px;
    margin-right: 10px;
    /* shorthand: T R B L */
    margin: 10px 10px 10px 10px;
    /* shorter: TB LR */
    margin: 10px 10px;
    /* shortest: TBLR */
    margin: 10px;
}

The margin property defines

spacing around an element

(can be set in px, em, or %)

If you set the width of an element, you can use margin: 0 auto; to

horizontally center it

(this is like setting text-align:center; for a text element)

If we were to use the column now, we’d find one big problem:

  • ...

these columns are acting like rows

o_O

.column1 {
    width: 60px;
    margin-left: 20px;
    float: left;
}

We want our columns to float, which defines

the horizontal flow

(commonly set to left, right, or none)

If you look closely at our columns now, you’ll see another problem:

we have 12 gutters, not 11

.__.

How do we remove a single gutter

when all of our columns have a left margin of 20?

The answer is not obvious but is very simple:

<div class="row">
   <!--columns-->
</div>
.row {
    margin-left: -20px;
}

We need a row to contain all the columns

with a negative left margin of 20 px

(the negative margin pulls the row to the left, removing 1 gutter for a perfect alignment)

Children are always affected by their parent's transform, or

position, rotation, and scale

(even if the children have their own transform)

.clearfix {
    *zoom: 1;
}
.clearfix:before, .clearfix:after {
    display: table;
    content: "";
}
.clearfix:after {
    clear: both;
}

Always add a clearfix to a floated element's parent

to fix their width & height

(by default, a floated element has no size!)
