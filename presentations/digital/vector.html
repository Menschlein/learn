Vector Graphics

are geometric data structures

Vector graphics are defined as

geometric primitives

(such as points, lines, curves, and polygons)

Geometry is a kind of math, and all vector graphics are

based on mathematical equations

Creating vector graphics

requires no math!

(yay?)

Unlike raster graphics, using math makes

vector graphics scale perfectly

Scaling is the act of

changing an object’s size

(without changing the object itself)

Think of it in terms of

a balloon

If you blow 2x as much air in, it's still the same balloon

only 2x bigger

If you take 1/2 of the air out, it's still the same balloon

only 1/2 as big

The strength of vector graphics is

perfect scaling

(they work like the balloon)

We use vector graphics much more

than raster graphics

Vector graphics are used for

Logos

Vector graphics are used for

User Interfaces

When we scale type up

it remains crisp and clean

When we scale an image up

it looks horrible and jagged

This is because one is vector and the other raster

A raster graphic is

a collection of millions pixels

(each with its own color)

Compared to millions of bits of color,

A vector graphic is exceedingly simple:

a collection of points, a fill, and a stroke

A point is defined as

a 2D location on the screen

(sometimes called an anchor)

A vector graphic

rarely has more than a few hundred points

A vector shape can be closed, where all points

are connected twice

A vector shape can be open, where any point

may only be connected once

Always used closed vector graphics

when drawing polygons or filled shapes

Each point may also have controls, which are like

line magnets

(they pull the line towards themselves)

A fill is

the color or gradient

that fills a shape

(made by the points, it can be invisible)

A stroke is

the color and thickness

of the line that forms the shape

(it can be invisible)

Vector graphics are almost always

simpler & smaller

when compared to raster graphics

Use vector graphics when you need something

to look right no matter where or how it's seen
