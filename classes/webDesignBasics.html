Introduction its about to get real wierd

Intent

This class will take a designer's approach to learning front-end web design and development, and seeks to guide each student to a place where they can comfortably combine design and code.

Prerequisites

Algebra and/or Geometry (we'll work with what you remember)

Although this class is primarily studio focused, with scheduled lectures and studio activities, it also includes elements from science and math. These can be intimidating areas, but remember, we are learning as artists and designers, not scientists.

Check it out: Students with an understanding of any of the basic areas are encouraged to diverge from the course structure and speak with me about an independent study.

The Satisfactory Single-page
Summary

Single-page information / marketing site for a band or musician of your choice.

What You Can Use

HTML, CSS

Timeline

2 / 15 weeks

Example

Your single-page should look like this!

Requirements
Page width of 940px
Full-width page introduction (the hero)
Hero biggest header
Hero biggest subheader
Row with 3 evenly-spaced columns of information (the sidekicks)
Sidekick 1: Bio (medium header + body copy at least 50 words)
Sidekick 2: Music (medium header + 2 records)
Sidekick 2: Record 1 (medium subheader + short description + list of at least 5 songs)
Sidekick 2: Record 2 (medium subheader + short description + list of at least 3 songs)
Sidekick 3: Events (medium header + table with name, location, date of at least 6 events)
Row with medium header and 1 full-width column (the members)
Members column contains a row with 1 evenly-spaced column per member
Each member has smallest header + bio + background color
Commented start and end of each hero, sidekick, and member
HTML organized with comments
CSS organized with comments

The Masterful Multi-page
Summary

Multi-page navigatable information / marketing site for a band or musician.

What You Can Use

HTML, CSS

Timeline

2 / 15 weeks

Requirements
Pages container width 940px and centered horizontally
Overview / Home page
Overview contains hero introduction, bio, events, and members
Music page
Music page contains horizontal list of album buttons as in-page links to full album
Each album has description, genre, all songs with name and track length, and a link to return to list
Navigation bar with links to all pages
Navigation bar links are relative
Navigation bar is fixed in screen
Navigation bar contains logo
Images on at least 1 page
Uses rounded corners
Bars use light box shadows
Bars, buttons, and anchors use soft gradients
Footer with copyright on all pages
HTML organized with comments
CSS organized with comments

The Fantastic Fluid-page
Summary

Mobile-ready fluid information / marketing site for a band or musician.

What You Can Use

HTML, CSS

Timeline

1 / 15 weeks

Requirements
Uses multi-page as base
At page min width 1200px: Rows + Columns, container width 1170px, column width 70px margin-left 30px
At standard size: Rows + Columns, container width 940px, column width 60px margin-left 20px
At page min width 768px and max width 979px: Rows + Columns, container width 724px, column width 42px margin-left 20px
At page max width 767px: single column / vertical stacked layout (i.e. no float) with fluid 100% width and minimum width 280px
Base font-size 16px
Uses at least 1 non-web-native font through @font-face

By Your Powers Combined...
Summary

Dynamic, mobile-friendly, multi-page informational / marketing site for a band or musician.

What You Can Use

HTML, CSS, Javascript

Timeline

6 / 15 weeks

Requirements
Overview / Home page
About page with description, genre, members, and bios
Discography page with at least 3 albums
Events page with at least 10 locations
Navbar with relative page links
Footer with copyright and page links
Logo in each page
Uses at least 2 fonts but no more than 4 font-families
Generative art / design using canvas in at least 1 page
Generative element is background / secondary to all else
Non-white color background in each page (can be generative element or image pattern)
Works with script disabled
Includes noscript warning where necessary
HTML organized with comments
External CSS organized with comments
Javascript commented
HTML passed W3C validation with less than 5 errors
